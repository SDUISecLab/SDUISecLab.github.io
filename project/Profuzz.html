Profuzz

Profuzz is a fuzzing platform which focuses on discovering vulnerabilities in protocol design and implementation. Although fuzz testing has already been a well-researched area, existing protocol fuzzers are not efficient enough. Grammar-aware fuzzing, learning-based state machines discovery, coverage-guided seed selection, fuzzing combined with sysbolic execution are still research highlights.
