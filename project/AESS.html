AESS

The AESS (Android EcoSystem Security) project implements a large-scale comprehensive measurement of the security of the Android ecosystem. This project is based on thousands of firmware from hundreds of vendors. Our goal is to provide new knowledge of the Android ecosystem security.

Ongoing Work

  • Android Deep Learning Applications Security Measurement
  • Android Firmware Ecosystem Security Measurement
  • Dynamic SDK Identification
